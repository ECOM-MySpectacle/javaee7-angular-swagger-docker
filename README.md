Simple container for JavaEE7 demonstration.

Description
===========

The Wildfly JavaEE7 serves :
* a RESTFul service
* a simple AngularJS application
* and the Swagger UI

Usage
=====

First, install Java (JDK), Maven and Docker.

 ./build.sh
 ./curl.sh

Exercices
=========

* Gatlings
* Hawkular
* Opentracing
* REST Testing with http://rest-assured.io/
